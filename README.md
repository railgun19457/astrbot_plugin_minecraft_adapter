# AstrBot Minecraft 适配器

连接 Minecraft 服务器和 AstrBot，实现消息互通和服务器管理功能。

> [!note]
> 孪生项目:[AstrBot Adapter](https://github.com/railgun19457/AstrBotAdapter)
> Minecraft插件，提供`WebSocket Server`和`REST API`
